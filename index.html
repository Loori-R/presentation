## DOM & Events

What is DOM?

  • Document Object Model.
  • The HTML represented as an object. 
    
							document.getElementById('id').innerHTML
  • Uses dot notation to access HTML elements.
  • We have used a little bit of it already.

What can we do with DOM?

  • Add and Remove HTML elements.
  • Modify CSS properties on HTML elements.
  • Trigger code when events happen on an element.
  • Change the innerHTML of elements.

DOM Events

  • Events trigger when certain criteria are met.
  • Such as a click, mouse over, key press, on load.
  • You can tie these events to functions that you create that will be run when the event triggers.
